# Star Technology — русский перевод (FTB Quests)

Русская локализация квестов и текстов модпака **Star Technology** (1.20.1, THETA-1-HOTFIX-3)
для мода **FTB Quests** через **ftbquestlocalizer**.

## Установка

Скопируйте `ftbquestlocalizer/lang/ru_ru.json` в:

```
<инстанс>/minecraft/kubejs/assets/ftbquestlocalizer/lang/ru_ru.json
```

Затем в игре: **Настройки → Язык → Русский**.

## Как сделан перевод

Перевод выполнен автоматически через OpenAI API (`gpt-4o-mini`) с последующей
ручной проверкой. При переводе сохранены §-коды форматирования, `\n`-переносы,
не переведены технические аббревиатуры уровней GregTech (LV/MV/HV/EV/IV/LuV/ZPM/UV/UHV и т.д.).
